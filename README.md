## 功能介绍

- 🤖 **实时开荒全家桶**：开荒太累？交给小助手！
    - **自动剧情**：跳过、分支选择、加速播放，剧情党和跳过党都沉默了。
    - **辅助战斗**：普攻、技能、终结技、连携技，你只管跑位，剩下的交给 MAA。
    - **自动拾取**：路边的野花（和掉落物）你不要采，我来采。
    - **便捷操作**：快速传送确认、自动长距滑索，老普再也不用担心我的手指。
- 🧩 **拼图大师**：解谜太烧脑？一键自动通关，支持单次、重复和演示模式，智商在线（指小助手）。
- 💰 **倒卖小天才**：自动识别高利润物资并购买，跨地区贸易从未如此简单。
- 📐 **蓝图搬运工**：批量导入蓝图，支持智能解析分享码，基建狂魔必备。
- 📅 **日常一条龙**：一键领取每日奖励，百科红点强迫症福音，会话消息自动清理。
- 🎮 **一键启动**：PC 端和安卓端都能自动启动并进入游戏，懒人福音。
