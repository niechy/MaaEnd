## ✨ 功能一览

- 🤖 **实时开荒辅助**：终极挂机神器！自动剧情、辅助战斗、自动拾取、快速传送，开着就去干别的事吧~
- 🧩 **自动解拼图**：AI 秒解各种谜题，脑子？不需要！支持单次/循环/演示模式 😎
- 🎁 **基建一条龙**：自动领产物、补货、收发线索，挂机收菜美滋滋~
- 📅 **日常自动化**：邮件、通行证、每日任务奖励一键全领！强迫症福音 ✅
- 💰 **半自动倒卖**：智能识别利润空间，低买高卖赚差价，财富自由指日可待（划掉）💸
- 📐 **蓝图搬运工**：批量导入蓝图，智能解析分享码，建筑党效率拉满！
- 🚚 **送货任务**：自动接取转交，跑腿不用愁~
- 📦 **模拟空间**：奖励自动领，不漏一分！
- 🤝 **拜访好友**：生产助力、情报交流全自动~
- 🌾 **简制手册**：一键领取所有奖励！
- 📖 **百科已读**：一键消灭红点，强迫症必备！
- 💬 **Baker 对话**：自动回消息，社恐救星~
- 🔨 **批量制作**：简易制作一键搞定！
- 💻 **多端适配**：PC 端 + 安卓端，陪你一起摸鱼~ 🐟
